JWT Tutorial

This is part of the JWT tutorial from https://auth0.com/blog/authentication-in-golang/. I have formatted this app to give a simple demonstration for a blog post on JWT. There are a few URLs that you can visit:

  1. Get Token => This page will generate and return a JWT that can be used to access the Products page.
  2. Products => A random list of products that can only be accessed by including the JWT in a header.
  3. Beyonce! => I wanted to learn how to serve images.

If you want to see more JWT details, look through some of the commit history of this repo or build your own app!
